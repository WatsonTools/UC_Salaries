# Academic Salary Growth Gap Web App

This is a static web app that can be deployed directly to Netlify.

## Files

- `index.html`
- `styles.css`
- `script.js`

## What it does

- Lets the user choose an academic position and start year
- Compares the actual salary path against a counterfactual salary path that keeps pace with national average wage growth
- Reports:
  - current annual salary gap
  - cumulative missed earnings since the selected start year
  - actual salary growth
  - benchmark growth at national wage growth
- Shows both a chart and a year-by-year table

## Deploy to Netlify

1. Open Netlify
2. Create a new site from drag and drop, or connect a repo
3. Upload the contents of this folder

No build command is needed because this is a plain static site.
